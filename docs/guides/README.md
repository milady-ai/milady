# Developer Guides

Comprehensive documentation for developing with Milaidy/ElizaOS.

## Available Guides

### [Plugin Development Guide](./plugin-development.md)
Complete walkthrough for creating, testing, and publishing plugins. Covers:
- Plugin architecture and lifecycle
- Actions, Providers, Services, Routes
- Event handlers
- Testing strategies
- Publishing to npm

### [Skills Documentation](./skills.md)
Guide to markdown-based skill extensions. Covers:
- Skill structure (SKILL.md)
- Frontmatter configuration
- Adding skills to agents
- Best practices

### [Contributing Guide](./contributing.md)
How to contribute to Milaidy/ElizaOS. Covers:
- Development environment setup
- Project structure
- Code style and conventions
- Pull request process
- Community resources

### [Local Plugin Development](./local-plugins.md)
Developing plugins without npm publishing. Covers:
- Workspace discovery
- Configuration-based loading
- Hot reloading
- Debugging
- Migration to npm

### [Registry Guide](./registry.md)
The plugin registry and ecosystem. Covers:
- Discovering plugins
- Using plugins
- Plugin manifest format
- Submitting plugins
- Categories and naming
